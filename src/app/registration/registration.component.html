Frontend

For the frontend, I have utilized HTML and CSS to enhance the user interface, creating a visually appealing and professional look for the home page. The design incorporates a corporate color scheme to maintain a cohesive branding experience. On the home page, users are provided with the option to register by filling out a registration form. The form includes fields such as Name, Employee ID, Gender DOB, Designation, Department and Appointment Date

Backend

The backend project, on the other hand, handles the logic and functionality of the application. It includes server-side code written in Python using Django, a popular web framework. The backend project manages tasks like data storage,logic, authentication, and authorization.

settings

API

With the API in place, the frontend can make HTTP requests, such POST to interact with the backend. For example, the frontend can send a post request to store data from the backend, or a POST request to send new data for storage.
